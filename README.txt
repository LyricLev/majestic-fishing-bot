# Majestic RP Fishing Bot 🎣

Автоматический макрос-бот для рыбалки, работающий на основе компьютерного зрения.

## Технологии
* **Python**
* **OpenCV** (анализ триггерных зон экрана)
* **CustomTkinter** (современный графический интерфейс)
* **PyDirectInput** (низкоуровневая эмуляция нажатий для обхода блокировок ввода в игре)

## Установка исходного кода
Если вы хотите запустить бота из исходников, установите зависимости:
```bash
pip install pydirectinput opencv-python numpy pyautogui pillow customtkinter
