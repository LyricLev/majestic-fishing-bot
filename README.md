# 🎣 Majestic RP Fishing Bot v1.0

> **Автоматический макрос-бот для рыбалки на Majestic RP (Alt:V), работающий на основе компьютерного зрения (OpenCV).** > Скрипт анализирует выбранную область экрана, распознает начало мини-игры и имитирует нажатия клавиш на низком уровне через DirectX.


## ✨ Основные фичи
* **Полный GUI интерфейс:** Красивое темное окно на `CustomTkinter` для управления и настройки.
* **Обход блокировок ввода:** Использование `PyDirectInput` вместо обычного `PyAutoGUI` — нажатия идут на уровне «железа», игра их не блокирует.
* **Умный анализ:** Бот не спамит кнопки вслепую, а ждет поклёвку по цвету пикселей через OpenCV.
* **Humanization (Анти-античит):** Все задержки между кликами и реакцией на поклёвку полностью рандомизированы, чтобы имитировать действия живого человека.
* **Своя утилита калибровки:** В комплекте идет `screen_helper` для быстрой настройки под любое разрешение (включая UltraWide 3440х1440).


## 🚀 Как запустить (Для пользователей)

Если вы не хотите устанавливать Python, просто скачайте готовый архив в разделе **[Releases](https://github.com/LyricLev/majestic-fishing-bot/releases)** на этом GitHub.

### 📋 Важные требования для работы:
1. 🪟 **Режим игры:** GTA 5 должна быть запущена в режиме **«В окне без рамки»** (Windowed Borderless). В полноэкранном режиме бот будет видеть только черный экран!
2. 🛡️ **Права администратора:** Файл `gui.exe` (или консоль при запуске `.py`) нужно обязательно запускать **От имени администратора**, иначе Windows не пропустит клики внутрь игры.


## 🛠️ Калибровка под свой монитор

Поскольку у всех разные экраны, перед первым запуском нужно указать боту, куда именно смотреть:

<details>
<summary><b>📖 Пошаговая инструкция по настройке координат (Нажми, чтобы раскрыть)</b></summary>

1. Запустите `screen_helper.exe` (или `python screen_helper.py`). Откроется черная консоль.
2. Зайдите в игру и начните рыбачить вручную.
3. Когда появится мини-игра (шкала рыбалки), наведите курсор мыши в **левый верхний угол** этой шкалы.
4. Свернитесь и посмотрите в консоль хелпера — запишите первые цифры `X` и `Y`. Это ваши **X1** и **Y1**.
5. Вернитесь в игру и наведите мышь в **правый нижний угол** шкалы. Запишите координаты — это **X2** и **Y2**.
6. Закройте хелпер, откройте `gui.exe` и вбейте полученные 4 цифры в соответствующие поля. Настройки сохранятся автоматически!
</details>
